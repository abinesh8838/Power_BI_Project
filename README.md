# Nike Shoes Sales Analysis – Power BI
## Project Overview

  This project analyzes Nike shoes sales data to uncover insights around product performance, revenue trends, and customer preferences.
The dashboard is built using Power BI and focuses on helping stakeholders understand what sells, when it sells, and why it sells.

The analysis is suitable for retail, e-commerce, and business analytics use cases.

## Objectives

* Analyze overall sales performance of Nike shoes

* Identify top-selling products and categories

* Understand sales trends over time

* Evaluate the impact of pricing and discounts

* Provide actionable business insights through interactive visuals

## Dataset Description

The dataset contains Nike shoe-level sales information, including:

* Product name / model

* Category (Running, Training, Lifestyle, etc.)

* Price and discounted price

* Units sold

* Total revenue

* Date of sale

* Customer ratings (if available)

 Data Source: Public dataset (Kaggle / open data source) 
 
 File Type: CSV (imported into Power BI)

## Tools & Technologies

* Power BI Desktop

* Power Query – Data cleaning & transformation

* DAX – Calculated measures and KPIs

* GitHub – Version control & project sharing

## Data Cleaning & Transformation

The following steps were performed in Power Query:

* Removed duplicate and null records

* Standardized column names and data types

* Converted date fields into proper date format

* Created calculated columns for revenue and discounts

## Key Measures (DAX)

Some important measures created:

* Total Sales

* Total Revenue

* Average Selling Price

* Units Sold

* Discount %

* Year-over-Year Sales Growth

## Key Insights

* A small number of models contribute to a large share of total revenue

* Discounted products show higher sales volume but lower margins

* Seasonal spikes are observed during promotional periods

* Certain mid-priced shoes outperform premium products in volume
